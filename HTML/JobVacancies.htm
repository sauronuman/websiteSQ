Job Vacancies

On this page, you will find information on job specifications as a web developer as well as how to kick-start your application if you want to apply to work for us. Here is a list of requirements and responsibilities as a web developer.

    Web Developer Requirements

    Here is a list of requirements that a web developer must be able to fulfill to be successful.

    • Bachelor’s degree in Web development or any related field or relevant experience in web development
    • Adequate knowledge and previous experience in programming
    • Competent in using HTML, CSS, JavaScript and MySQL
    • Can be a dedicated team player
    • Be able to thrive in a work environment that can be fast-paced
    • Competent skills in both written and verbal communication
    • Be able to demonstrate sufficient knowledge in programming language
    • Must be able to use technical terminology where appropriate
    • Must be able to develop and express ideas clearly
    • Be able to solve complex problems
    • Be able to make independent choices that are most appropriate

 

Web Developer Responsibilities

As with any job, there are some responsibilities involved. Here is a list of some responsibilities as a web developer.

  • Designing, building, testing and possibly maintaining a website or software application
  • Working with teams to resolve any conflicts, develop content criteria, prioritize requirements and find and agree o solutions
  • Performing and directing any website updates
  • Writing, designing and editing website content and (or) directing other team members that are producing content for a website
  • Backing up files from websites to local directories so that they can be recovered
  • Identify errors through testing or customer feedback and rectifying any errors or referring them to relevant employees to be rectified
  • Making sure that code meets industry standards and is compatible with a range of web browsers, operating systems and devices
